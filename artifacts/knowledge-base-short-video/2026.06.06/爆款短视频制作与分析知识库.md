# 爆款短视频制作与分析知识库

本文档由三个源文件整理而成，用于知识库导入。内容分为四个模块：爆款视频制作流程、团队作战 SOP、短视频画面设计方法论、爆款视频分析回答模板。

## 0. 知识库使用说明

- 用于回答“爆款视频怎么做”“视频怎么拆解”“拍摄前怎么准备”“画面怎么设计”“团队如何协作”“怎么写分析反馈”等问题。
- PDF 中含有大量水印和思维导图截图。正文已尽量抽取并结构化；对纯图片流程图中可确认的内容，整理为流程要点，不把不可辨认的小字扩写为事实。

## 1. 爆款视频制作流程总览

### 1.1 两种作战模式

爆款短视频从 0 到 1 的准备和制作流程分为两种情况：

| 模式 | 适用对象 | 核心提醒 |
|---|---|---|
| 个人作战 | 没有团队、独立完成短视频创作的人 | 重点提升拆片、拆稿、拆人群能力，短期内快速提高内容制作能力。 |
| 团队作战 | 有编导、操盘手、摄影、剪辑等协作角色的团队 | 短视频拼的是质量和效率，团队的“量”未必打得过小团队的“质”，必须用 SOP 提升协作效率。 |

### 1.2 爆款内容的基本工作链路

```text
账号定位 -> 选题库搭建 -> 拆片拆稿拆人群 -> 拍摄准备 -> 拍摄执行 -> 剪辑包装 -> 发布测试 -> 数据复盘 -> 优化迭代
```

### 1.3 个人作战重点

个人作战的核心不是一次性建立复杂团队流程，而是先把“选题判断、内容拆解、画面最低标准、个人素材库”做扎实。

关键工作：

- 搭建选题库：从对标账号、赛道优质视频、真实数据中收集选题。
- 做选题判断：用收集来的选题分析用户需求、二级/三级选题延展空间、内容可执行性。
- 梳理知识体系：课程研发、账号核心价值观、人设思维底色，决定账号三观和行动指令。
- 梳理人生体验：按时间线写清人生事件，再从事件中的具体感受提炼观点。
- 补齐信任背书：收入证明、工作证明、证书、合影、生活切片、游玩照片、纪念品、特殊事件物品和影像素材等。
- 做服化道准备：拍摄环境要干净整洁，空间要大、纵深足、明亮。

### 1.4 内容方向核心

以女性成长赛道示例，内容方向可以概括为“提升能量、提升自信力”。用户需求底层逻辑是希望被看见、被理解、被支持，同时需要更理性地分清问题。

表达观点时要注意：

- 观点输出要明确、坚定。
- 使用场景要清楚具体。
- 只要是知识体系、观点和三观，就要减少模糊摇摆。
- 选题和观点输出要有鲜明判断，否则用户难以形成记忆。

### 1.5 拆片、拆稿、拆人群

拆片拆稿的目标不是模仿表面，而是快速拆出结构和用户需求。

重点拆解：

- 选题：洞察了什么需求，提供了什么解决方案。
- 人群：目标用户是谁，用户需求量有多大。
- 结构：有没有钩子、骨架、情绪刺点。
- 表达：内容是否抽象，是否易懂。
- 信息密度：是否持续给新信息。
- 含金量：是否有新鲜感、价值和认知增量。
- 画面：开头画面是否比常规视角更吸引人。

## 2. 团队作战 SOP

### 2.1 四周阶段目标

| 周期 | 工作内容 | 阶段目标 |
|---|---|---|
| 第一周 | 制定 IP 方向、IP 选题策划、选题和内容梳理、制作流程方案梳理、建立选题库、筛选第一轮选题、制定排期计划和更新计划 | 与 IP 保持紧密沟通，完成账号方向和内容基础设施。 |
| 第二周 | 拍摄方案制定和执行、拍摄第一轮选题、同步更新优化选题库 | 与 IP 磨合减少分歧，优化排期和更新时间，开始拍出成片效果并复盘优化。 |
| 第三周 | 持续更新，获得测试反馈 | 重点发现问题，反推工作流程，优化流程和选题。 |
| 第四周 | 基于数据反馈复盘优化 | 让账号通过测试期，进入成长期。 |

### 2.2 第一周：IP 方向定位

IP 会面要对 IP 做系统了解，完整记录：性格、职业、履历、成就、擅长点、增长点、闪光点。会面前准备好工具包和记录工具。

IP 定位需要梳理：

- IP 变现方向：变现方式、方法、阶段打法、配套人员。
- IP 内容方向：赛道、调性、人设、垂直领域、内容输出方法、差异化。
- IP 选题：对标账号、话题、同质化优质选题、差异化优质选题。
- 孵化周期：测试阶段、成长阶段、冲刺阶段。

### 2.3 第一周：内容和选题策划

选题制定要围绕 IP 感兴趣且擅长的方向，搭建选题库并明确收集选题的方法。

内容呈现形式要匹配 IP 形象和特质，包括拍摄形式、内容形式、剪辑形式、人设展现形式。

内容要素占比包括两层：账号内容占比，即营销内容、涨粉内容的比例；视频内容占比，即专业内容、泛话题内容、热点内容的比例。

选题和内容梳理要做到：

- 从选题库确定第一轮选题。
- 同时确定备选选题。
- 与 IP 沟通确认选题可用。
- 如果是采访类型，要准备采访问题。
- 如果是文案型，要及时与达人对齐选题产生的内容。
- 内容要细化到段落。

### 2.4 采访式内容生产流程

```text
第一次采访 -> 生成选题 -> 第二次采访并正式拍摄 -> 整理逐字稿 -> IP 反馈和背诵 -> 第三次采访 -> 剪辑并寻找出彩点 -> 必要时第四次采访
```

执行要点：

- 第一次采访不拍摄，主要收集 IP 感兴趣话题和擅长话题。
- 从第一次采访内容中生成选题。
- 第二次采访针对生成选题开始，并进入正式拍摄。
- 第二次采访后整理完整内容稿，由操盘手排序、修改。
- 修改后的逐字稿发给 IP，IP 反馈后继续修改并背诵。
- 第三次采访针对 IP 背诵的逐字稿，重点看表达状态和金句发挥。
- 剪辑第三次采访内容，从成片中寻找出彩点，记录下来与 IP 分析能否放大和延展到全片。
- 如果成片还有提升空间，再进行第四次采访。

### 2.5 账号搭建与养号

账号注册需要物料：专用运营手机一个、电话号码一个、未实名身份证一张。账号注册由编导负责。注册后不要退出登录，准备开始养号。

养号流程：

- 进入抖音后不要搜索任何关键词和账号名称。
- 通过刷内容让账号获得兴趣标签。
- 非本赛道内容点击“不推荐”或“不喜欢”。
- 本赛道视频多看，多进直播间，多进评论区有效互动。
- 禁止发布“666”这类无效评论。
- 养号周期为 3-5 天。
- 养号时间段要符合正常人类行为，把新号当成自用号去刷。
- 养号期间刷视频、进直播间、互动等行为都要符合真人行为。

还需要确认：账号名称、简介、实名制问题、企业号注册问题、认证流程。

### 2.6 拍摄前准备

前期拍摄准备包括：场地、服装、妆容、内容、灯光和机位、收音、道具、提问问题顺序和流程、提问方式方法，以及与 IP 沟通整体方案、选题内容和拍摄细节。

拍摄方案制定和执行包括：

- 确定拍摄器材和灯光。
- 确定拍摄场地。
- 确定达人呈现效果。
- 正常开始拍摄。
- 正式拍摄时，摄影师不允许在拍摄过程中自行更改相机参数、灯光位置、灯光参数。

### 2.7 素材交接和现场执行

素材交接：

- 拍摄完毕后，摄影师把素材交给剪辑。
- 剪辑第一时间保存素材，并进行素材转代理。
- 拍摄当天要对全部机位素材完成对轨。

现场执行：

- 采访或素材跟拍时，及时调整 IP 输出内容和状态。
- 现场速记 IP 输出内容。
- 现场速记规范为：时间 - 话题 - 大致内容 - 金句 - 观点。
- 拍摄结束后，在现场速记基础上，对照时间节点和现场记录文字，安排素材剪辑优先级。
- 梳理素材，确定粗剪内容。
- 精细化梳理素材，确定精剪内容。

### 2.8 剪辑执行流程

剪辑执行分为素材整理、挑选素材、粗剪、精简包装。

素材整理和分类：

- 查看素材。
- 编导用飞书文档把逐字稿段落内容同步给剪辑。
- 所有改动，剪辑和编导同时在同一个飞书文档中执行修改。

挑选素材：

- 编导在飞书文档中找到第一轮素材。
- 第一轮素材通常是 IP 对一个话题的完整宽泛叙述。
- 剪辑通过飞书文档摘取素材准备剪辑。
- 摘取素材后，编导第一次筛选文案。
- 剪辑等待文案筛选时，进行多机位对轨、调声音、调色。

粗剪：

- 编导在筛选好的内容基础上进行第一轮筛选。
- 删掉废话、重复语段、无聊点。
- 编导标注粗剪中要筛掉的句子。
- 剪辑按修改好的文案排序剪辑。

精简和包装：

- 编导在第一轮删改文案基础上进行第二轮修改。
- 筛掉不影响视频整体的文案。
- 提炼金句。
- 调整文案位置。
- 一切以增加信息密度为主。
- 更换过的位置，编导要在备注处标明。

### 2.9 发布后复盘

掉人点复盘方法：

- 每条视频发布后第二天，在数据分析后台找出停留曲线。
- 将停留曲线与大盘平均水平对比。
- 低于大盘的地方是必须优化的地方。
- 掉人点明显大于自己平均值的地方是必须优化的地方。

优化顺序：

```text
剪辑节奏 > 音效 > 特效 > 音乐 > 拍摄 > 服化道 > 人物状态 > 文案
```

执行方法：在关键掉人节点和必须优化的位置，提前两秒开始分析；按上述顺序逐项排查并改正；如果排查到最后已经没有优化空间，则直接删掉这段内容。

## 4. 短视频画面设计方法论

# 短视频画面设计方法论

> **核心判断**：画面设计不是把画面拍好看，而是让观众在一眼之内知道：这里有什么欲望、这是谁的世界、这件事为什么值得看。
>
> **一句话原则**：能拍出来的，就别说出来；能用空间、物品、动作和时间让观众感受到的，就不要只用语言解释。

---

## 1. 画面设计的底层目标

短视频画面设计至少服务于四个目标：

| 目标 | 说明 | 常见问题 | 设计方向 |
|---|---|---|---|
| **激活欲望** | 让观众第一眼看见自己想要的东西 | 画面中心是人脸，而不是用户想要的生活 | C 位留给欲望、结果、场景或产品带来的改变 |
| **交代身份** | 空间和物品说明“你是谁” | 背景是一堵墙，没有身份信息 | 用环境、陈列、服装、道具建立身份感 |
| **制造情绪** | 光线、色彩、构图、景别影响观众感受 | 画面干净但没情绪 | 先确定希望观众获得的感觉，再反推视觉元素 |
| **提高信息效率** | 画面承担说明、补充、证明功能 | 所有信息都靠口播，视觉浪费 | 让画面承担证据、场景、对比、演示 |

对应课程来源：用画面讲故事、用户欲望分层、品牌美学探索。

---

## 2. 第一原则：欲望永远比产品大一号

### 2.1 不要只拍产品，要拍产品之后的生活

用户欲望分层里有一个关键法则：**不要卖产品本身，卖产品之后的那个你。**

| 账号/产品 | 低效画面 | 高效画面 |
|---|---|---|
| 卖车 | 车身、内饰、参数 | 开车去海边、露营、接孩子、回老家 |
| 卖化妆品 | 瓶子特写、成分表 | 涂完之后自信的笑、约会前的状态、被夸气色好 |
| 卖课程 | PPT、讲师大头、课程表 | 学完后升职、接单、表达更清楚、人生改变的场景 |
| 卖服装 | 平铺、挂拍、面料说明 | 穿上后参加聚会、旅行、被人夸、状态变好 |

**画面设计公式**：

```text
产品/能力 -> 使用动作 -> 生活结果 -> 身份/关系/自我实现
```

不要停在“这个东西很好”，要继续往上问：
- 它让用户成为什么样的人？
- 它让用户进入什么样的生活场景？
- 它会改变用户和谁的关系？
- 它能让用户被怎样看见、被怎样夸赞？

### 2.2 画面 C 位留给欲望，不是留给你

很多口播视频的问题是：创作者的大脸占满画面，产品、场景、证据都缩在角落。这样画面传递的信息是“看我”，而不是“看你想要的东西”。

更有效的结构是：

1. **前 3 秒**：欲望怼脸。先给观众看最想要的东西。
2. **中段**：创作者进入画面，解释、证明、讲故事。
3. **后段**：人与欲望同框，建立人格背书和成交信任。

例如二奢账号：

```text
爱马仕包怼脸 -> 讲包的故事和细节 -> 人和包同框 -> 观众相信“这个人能帮我买到好包”
```

观众心理路径是：

```text
我想要这个 -> 这个东西不错 -> 这个人好像靠谱 -> 我愿意继续看/咨询/购买
```

---

## 3. 第二原则：空间不是背景板，而是一个世界

空间的处理和调度强调：拍摄空间不是“一堵墙”或“一个背景”，而是一个立体的三维集合。

空间至少包含：
- 材料与物质：墙面、桌子、窗户、植物、书架、产品、器具
- 光线：明亮、昏暗、自然光、人工光、冷暖光
- 纵深：前景、中景、背景，空间是否有层次
- 人与物的位置关系：人站在哪里，物放在哪里，观众先看见什么
- 场合属性：公开/私密、专业/生活、昂贵/便宜、热闹/孤独

### 3.1 空间的四个功能

| 功能 | 解释 | 例子 |
|---|---|---|
| **交代身份** | 你在哪个空间里，观众就会判断你是谁 | 书房=知识型，厨房=生活型，工厂=源头型，门店=经营者 |
| **影响情绪** | 光线、距离、开放度会改变情绪 | 暗光更私密，强光更直接，夕阳更温柔 |
| **塑造品质** | 空间质感影响产品和人的价格感 | 高客单产品不能长期在杂乱廉价空间里讲 |
| **承载故事** | 空间能说明过去、关系和变化 | 桌上的旧照片、磨损的工具、收纳箱里的旧物 |

### 3.2 空间设计检查表

拍摄前可以问 10 个问题：

- 这个空间能不能让观众立刻判断“我是谁”？
- 这个空间和我讲的内容是否一致？
- 背景里有没有多余的、会抢注意力的东西？
- 画面有没有前景、中景、背景？
- 光线是否服务于情绪，而不是只是“够亮”？
- 画面里最亮、最大、最动的东西，是不是我最想让观众看的？
- 空间里的物品是否能补充我的身份和专业度？
- 如果关闭声音，只看画面，观众能不能猜到这条视频大概在讲什么？
- 这个空间是可持续复用的吗？
- 它是否形成了我的账号记忆点？

### 3.3 直播间/门店画面的特殊原则

对门店、直播间、实体经营者来说，墙面、柜台、门头、桌面本身就是信息媒介。

关键做法：

- **门头不要像菜单**：只突出主打款和记忆点，不要堆满所有品类。
- **墙面不是装饰，是宣传单**：写清楚身份、故事、主打卖点。
- **食材/产品要师出有名**：用产地、工艺、人物故事给产品增加理由。
- **让用户不用被教就会点单**：最想卖的产品出现在最显眼的位置。
- **用局部布景制造传播素材**：一张桌子、一块布、一组道具，能反复拍几十条内容。

## 4. 第三原则：物品是视觉证据，不只是装饰

画面里的物品承担三种任务：

1. **证明你说的是真的**：奖杯、手稿、工具、订单、使用痕迹。
2. **传递你是谁**：书、茶具、衣服、乐器、工作台、电脑桌。
3. **激活观众欲望**：包、衣服、食物、车、房间、目的地。

### 4.1 道具选择公式

```text
观点/情绪 -> 需要观众相信什么 -> 哪个物品最能证明它 -> 物品如何进入动作
```

不要只问“画面里放什么好看”，而要问：

- 这个道具能不能证明我的专业？
- 它能不能让抽象概念变成具体画面？
- 它和我想表达的欲望有没有关系？
- 它能不能被使用，而不是只被摆放？

### 4.2 物品要尽量进入动作

品牌美学探索里提到“名词性思维”和“动词性思维”：状态型表达不如动作型表达有力量。

| 物品状态 | 更好的动作化画面 |
|---|---|
| 桌上有一本书 | 手指翻到折角页，旁边有写满批注的便签 |
| 桌上有一杯咖啡 | 冰块碰杯，咖啡倒进去，杯壁出现水珠 |
| 身上穿了一件衬衫 | 解开扣子，风吹动亚麻，链条斜挎包压出腰线 |
| 有一堆健康档案 | 一张张摊开：社保、大病卡、体检单、养老计划 |

物品静止时只是陈列；物品被使用时，才开始讲故事。

## 5. 第四原则：用动作代替状态，用细节代替大词

用画面讲故事的核心是：**Show, Don't Tell**。

### 5.1 不要说情绪，要拍情绪的证据

| 不要说 | 要拍 |
|---|---|
| 我很生气 | 摔门的手、颤抖的肩膀、被摔在桌上的杯子 |
| 她很孤独 | 她蹲在海边，用手指在沙子上画圈，又擦掉 |
| 他们很相爱 | 他帮她把头发别到耳后，她下意识躲了一下又笑了 |
| 这个人很靠谱 | 提前 5 分钟到场、电脑已打开、微信回“收到” |
| 生活很累 | 凌晨 2 点抱孩子在客厅走，窗外只有路灯亮着 |

### 5.2 细节比全景重要

全景负责交代，细节负责打动。

- 不要只拍整个房间，要拍桌上半杯冷掉的咖啡。
- 不要只拍办公室很忙，要拍揉皱的纸团和没关机的电脑。
- 不要只拍旅行很美，要拍鞋上的泥、手里的票、风吹乱的头发。

具象化表达的核心判断是：**人是被细节感动的，不是被大词感动的。**

## 6. 第五原则：画面信息量决定观众的心理时间

画面不仅塑造空间，也塑造时间。第二天-信息结构里提到：导演可以通过画面信息量控制观众的眼球转动速度，从而影响心理时间。

### 6.1 信息量越高，眼球移动越快

如果画面被分成两半，或者有多个视觉焦点，观众的视线会来回跳跃，心理时间会变快。

适合：
- 制造紧张感
- 表达复杂关系
- 展示对比
- 体现热闹、拥挤、繁忙

### 6.2 信息量越低，时间感越慢

单一主体、留白、稳定构图，会让时间慢下来。

适合：
- 表达平静
- 表达孤独
- 表达高级感
- 表达沉思、回忆、疗愈

### 6.3 时间感设计工具

| 工具 | 效果 | 适用 |
|---|---|---|
| 慢动作 | 拉伸时间，放大情绪 | 关键动作、情绪爆发、仪式感 |
| 延时 | 压缩时间，表现变化 | 做饭、搭建、装修、日出日落 |
| 抽帧/慢门 | 让动作变得迷离、风格化 | 情绪片、城市感、回忆感 |
| 定格 | 锁定瞬间，制造总结或回味 | 结尾、人物转折、命运感 |
| 叠化 | 让时间蔓延，制造记忆感 | 回忆、四季、旧地重游 |
| 跳切 | 省略过程，提高节奏 | 教程、vlog、快节奏口播 |
| 单一运动 | 稳定、秩序、沉浸 | 治愈、专注、仪式性动作 |

心法：根据自己的时间感，调整语气、情绪、剪辑节奏和每一个镜头设计。

## 7. 第六原则：风格不是滤镜，是感觉记忆

品牌美学探索把风格定义为：**观众看完你的视频，可能忘了具体内容，但会记得看你视频的那种感觉。**

风格来自五类感觉源头：

- 绘画：光影、色彩、构图
- 电影：叙事方式、镜头语言
- 音乐：节奏、旋律、情绪
- 文学：文字质感、叙事口吻
- 个人经历：你真实生活里的记忆和气味

### 7.1 风格的三个层次

| 层次 | 内容 | 风险 |
|---|---|---|
| **视觉风格** | 色调、构图、景别、剪辑节奏 | 容易模仿，也容易同质化 |
| **语言风格** | 说话方式、句式、口头禅、叙事口吻 | 需要长期表达积累 |
| **人格风格** | 你的价值观、情绪状态、生命经验 | 最难复制，也最有壁垒 |

视觉风格是入口，但不是终点。最高级的风格，是画面、语言、价值观和人格状态统一。

### 7.2 找形式支点，而不是复制表面

不要问“这个博主用了什么滤镜”，要问：

- 这个作品真正打动人的形式支点是什么？
- 是停顿？对称构图？第一人称视角？碎片化记忆？慢节奏？
- 这个形式支点能否迁移到我的内容里？
- 它和我的人格、行业、产品是否匹配？

例子：

| 参考对象 | 不要只学 | 要找形式支点 |
|---|---|---|
| 王家卫 | 昏黄滤镜 | 抽帧、慢门、暧昧距离、错过感 |
| 韦斯安德森 | 色彩鲜艳 | 对称构图、秩序感、童话式荒诞 |
| 罗大雄 | 低饱和生活感 | 时间记录者视角、低能量、平静细节 |
| 余文亮 | 粗糙镜头 | 普通人的生命力和不完美真实 |

## 8. 不同内容类型的画面设计策略

### 8.1 口播知识类

重点不是“背景好看”，而是减少理解成本。

建议：
- 画面干净，但不要空。
- 背景保留 1-3 个身份锚点，如书、白板、电脑、行业工具。
- 关键概念用手写板、屏幕、道具辅助说明。
- 每 10-20 秒给一次视觉变化：换景别、插入案例、切到手部动作。

适用概念：原子化结构、视频信息处理方法。

### 8.2 产品带货类

重点是“欲望前置 + 使用动作 + 结果画面”。

建议：
- 前 3 秒让产品或结果占据视觉中心。
- 不只拍产品，要拍使用前后、场景、用户状态。
- 高客单产品要重视空间质感；低价福利品可以强化密度、数量、抢购感。
- 每个卖点都尽量配一个可见证据。

### 8.3 生活 vlog 类

重点是“明线细节 + 暗线感觉”。

建议：
- 明线：吃了什么、去了哪里、做了什么、见了谁。
- 暗线：这是一种怎样的生活方式？让人羡慕、平静、自由、疗愈，还是热血？
- 多拍时间、物品、动作：凌晨、餐桌、路灯、手、风、收纳箱。
- 少用大词，多用可感知细节。

### 8.4 门店/实体经营类

重点是“让空间自己成交”。

建议：
- 门头只讲主打记忆点。
- 墙面承担故事、产地、老板身份、主打产品说明。
- 桌面和货架要为拍摄而摆，不只是为经营而摆。
- 每个可拍角落都应该能独立成为一条短视频素材。

### 8.5 人设 IP 类

重点是“空间、物品、动作共同说明你是谁”。

建议：
- 不要只靠自我介绍建立人设。
- 用固定空间形成记忆点。
- 用固定动作形成识别，如泡茶、翻书、整理工具、走进门店。
- 用固定物品形成身份锚点，如笔记本、老相机、样衣、产品箱。

## 9. 画面设计流程：从选题到拍摄

### Step 1：确定观众反应

先问：这条视频希望观众产生什么反应？

- 想要
- 羡慕
- 相信
- 被理解
- 被震撼
- 想模仿
- 想咨询
- 想转发

对应概念：内容思维。

### Step 2：确定欲望 C 位

问：这条视频里，用户真正想看的是什么？

- 是产品？
- 是结果？
- 是生活方式？
- 是某种身份？
- 是一个具体场景？
- 是创作者本人的状态？

### Step 3：设计空间

问：在哪里拍，最能让这件事成立？

- 书房、办公室、门店、工厂、街头、厨房、车里、卧室、旅途？
- 空间需要专业、真实、松弛、昂贵、热闹、孤独，还是私密？

### Step 4：选择物品和道具

问：哪些物品能证明、补充、激活欲望？

道具不是越多越好，而是越准确越好。

### Step 5：把状态改成动作

把“我很专业”“这个很好用”“生活很美”转化成动作证据。

```text
形容词 -> 具体行为 -> 具体物品 -> 具体场景
```

### Step 6：安排节奏与时间感

问：这条视频应该快还是慢？

- 快：高信息密度、剪辑跳切、多焦点画面。
- 慢：稳定构图、留白、少动作、低信息量。

### Step 7：检查风格统一

问：这条视频和我的账号整体感觉一致吗？

- 色调一致吗？
- 景别一致吗？
- 节奏一致吗？
- 说话方式一致吗？
- 情绪状态一致吗？

## 10. 拍摄前自检清单

### A. 欲望检查

- [ ] 观众前 3 秒能不能看到自己想要的东西？
- [ ] 画面 C 位给了欲望，而不是无意义的大脸或空背景？
- [ ] 产品之后的生活有没有被拍出来？

### B. 空间检查

- [ ] 这个空间能交代我的身份吗？
- [ ] 空间情绪和内容情绪一致吗？
- [ ] 背景里有没有干扰注意力的杂物？
- [ ] 有没有前景、中景、背景的层次？

### C. 物品检查

- [ ] 道具是否服务于观点，而不是单纯装饰？
- [ ] 物品有没有进入动作？
- [ ] 有没有一个能被观众记住的身份锚点？

### D. 动作检查

- [ ] 有没有把抽象状态拍成具体动作？
- [ ] 有没有至少 1-2 个能打动人的细节？
- [ ] 关掉声音后，画面还能不能传递部分信息？

### E. 风格检查

- [ ] 色调、构图、剪辑节奏是否统一？
- [ ] 这条视频是否有我的账号辨识度？
- [ ] 形式是否服务内容，而不是为了炫技？

## 11. 常见误区

### 误区一：把画面设计理解成“拍得高级”

高级感只是形式，不是目的。画面真正要解决的是：观众想不想看、看不看得懂、信不信、有没有感觉。

### 误区二：背景越干净越好

干净不等于有效。过度干净的背景会抹掉身份信息。好的空间应该是“有秩序的信息”，不是“没有信息”。

### 误区三：产品永远放最大

产品只是通往欲望的入口。很多时候，真正应该放大的不是产品，而是产品带来的状态、关系和生活。

### 误区四：盲目模仿爆款风格

别人的风格来自别人的人生。你可以学习形式支点，但不能复制人格气质。

### 误区五：每一帧都想塞满信息

画面信息量要服务节奏。该快时快，该慢时慢。密度不是越高越好，留白也不是越多越高级。

---

## 12. 一句话总结

画面设计的本质，是把“我要表达什么”翻译成“观众能看见什么”。

真正有效的短视频画面，不只是好看，而是：

```text
欲望清楚 + 身份明确 + 情绪准确 + 证据可见 + 风格统一
```

当画面能替你说明身份、激活欲望、补充信息、制造情绪时，你的视频就不再只是口播，而成为一个有世界感、有记忆点、有转化力的内容系统。

## 5. 爆火视频分析回答模板

### 5.1 原始模板

学员问爆款视频应该怎么分析
（1）我们一般是这么进行视频分析的，如果后续你想分析视频的话，你也可以按照这个流程顺序来：①.先看视频的选题（洞察需求+解决方案），你的目标用户对这个选题的需求量大不大，播放量的上限大概是多少，你有没有为用户提供解决方案，提供的解决方案是什么；②.接下来看视频整体内容是否抽象，会不会让用户难以理解；③.接下来看视频的信息密度；④.看视频的含金量，对用户来说这个视频有没有新鲜感、有没有价值；⑤.再看视频结构上的问题，视频有没有钩子、情绪刺点，视频的骨架，这部分建议看一下老师的课程《进阶篇：结构的精细化处理（钩子、骨架、情绪刺点）》。
（2）接下来我将会按照我们分析视频的流量，来分析你的【】这条视频。
①.视频选题，开头的“……”这句话的作用是（对观众来说）；咱们这条视频选题“……”的目标用户需求量还是很大的，抖音平台类似话题的播放量和点赞都非常高（最高赞有……），并且视频也为用户提供了解决问题的方案（……）；
②.在语言表达方向，相比于全程用专业词汇的视频而言，咱们这条视频内容也通俗易懂，讲清楚了……，这样大部分用户看完都能轻松理解，并且落地实施；
③.咱们这条视频信息密度也是够的，基本每10秒输出一个新信息，无废话，……，对观众而言一直有新的且有用的信息在刺激他，让观众能持续看下去；
④.视频的含金量也是够的，内容新鲜感强，a.打破 …… 的固有认知，b.并且为用户提供了解决问题的方案，……；
⑤.钩子上，开头这句话“……”打破了观众的认知差，提供了认知真相，让观众感觉能看到业内真实的专业可靠的知识，并且开头钩子的画面用的是……，这种……会比寻常视角更加吸引人。

### 5.2 可复用结构化模板

当学员问“爆款视频应该怎么分析”时，可以按以下结构回答：

```text
我们一般按这个顺序分析视频：
1. 先看选题：洞察了什么需求，提供了什么解决方案，目标用户需求量大不大，播放量上限大概是多少。
2. 再看内容表达：整体内容是否抽象，用户会不会难以理解。
3. 再看信息密度：是否持续输出新信息，有没有废话。
4. 再看含金量：对用户有没有新鲜感、价值感、认知增量。
5. 最后看结构：有没有钩子、骨架、情绪刺点，开头画面是否足够吸引人。

接下来我会按这个流程分析你的【视频主题/视频标题】。

1. 视频选题：开头“……”这句话的作用是……；这个选题的目标用户需求量……，类似话题数据……，并且视频提供了解决方案……。
2. 语言表达：相比全程专业词汇，这条视频更……，讲清楚了……，用户看完可以……。
3. 信息密度：基本每 10 秒输出一个新信息，……，能持续刺激观众看下去。
4. 内容含金量：新鲜感强，打破了……的固有认知，并提供了……解决方案。
5. 钩子与画面：开头“……”打破认知差，提供认知真相；开头画面使用……，比常规视角更吸引人。
```

## 6. 知识库问答索引

### 6.1 如果用户问“爆款视频怎么做”

优先回答：账号定位、选题库、拆片拆稿、拍摄准备、剪辑包装、发布测试、数据复盘。

### 6.2 如果用户问“团队怎么做短视频 SOP”

优先回答：四周阶段目标、IP 会面、选题策划、采访流程、账号搭建、拍摄执行、素材交接、剪辑流程、掉人点复盘。

### 6.3 如果用户问“怎么分析一个爆款视频”

优先回答：选题、用户需求、解决方案、抽象程度、信息密度、含金量、结构、钩子、情绪刺点、开头画面。

### 6.4 如果用户问“画面怎么设计”

优先回答：欲望 C 位、空间身份、物品证据、动作替代状态、细节替代大词、时间感、风格统一。

### 6.5 如果用户问“视频发布后怎么优化”

优先回答：第二天看停留曲线，对比大盘和自身平均值，从掉人点提前两秒分析，按剪辑节奏、音效、特效、音乐、拍摄、服化道、人物状态、文案的顺序优化。
